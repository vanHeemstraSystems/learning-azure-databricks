# Flow 01 — Factory Tour: The Complete Platform Overview

> **Goal**: Understand how all Azure Data Platform components relate to each other before diving into any single one.

## 🏁 Start Here

This flow is your **orientation lap** — one full circuit of the factory before you start drilling into individual stations.

## Steps

1. Read `articles/episode-01-welcome-to-the-factory.md`
2. Draw the metaphor map from memory (test yourself)
3. Provision a trial Azure Databricks workspace (Free Trial or PAYG)
4. Explore: Compute → Create All-Purpose Cluster
5. Explore: Data → Create a Delta table from sample data
6. Explore: Workflows → Create a simple one-task job

## ✅ You're Ready to Move On When You Can Answer

- What is the difference between the Control Plane and Data Plane?
- Name three Azure services in a typical data platform and their F1 roles
- What is the Medallion Architecture?
- Why do we use Delta Lake instead of raw Parquet?
